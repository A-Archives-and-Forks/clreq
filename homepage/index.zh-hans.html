简介

本小组旨在探讨 Web 和电子书在简体与繁体中文支持方面的不足，并记录与这些文字系统相关的需求。

我们的目标是在了解中文使用者需求的本地专家与 W3C 社区中制定 Web 标准的人之间搭建桥梁。讨论议题以差距分析模板为指导，成果反馈至文种矩阵。

征求协助

我们正在征集有关中文书写系统的信息。欢迎查看待解答的问题，了解您可以发挥专长的地方。

反馈

请使用GitHub Issue 提出中文支持相关问题、参与讨论或对文档提供反馈。您也可以了解如何使用 GitHub Issues。public-i18n-chinese 邮件列表用于发送通知摘要和会议记录，不用于技术讨论。

参与方式

您可以以不同方式参与：

  • 关注最新动态：订阅 public-i18n-chinese 邮件列表，以摘要形式接收相关议题。若同时希望关注中、日、朝（韩）文，请订阅 public-i18n-cjk。
  • 贡献内容：阅读并同意贡献者指南，在议题讨论中分享观点或提交 PR。
  • 投入更多：若希望成为参与者、编辑或主席，请联系 Richard Ishida 或 薛富侨，并参阅参与语言支持项目了解详情。
